# Landrop
Landrop is a simple Windows application that allows you to send files and folders to other PCs or smartphones in your local network or your Tailnet (Tailscale).
## Features
- Send files and folders
- access the Landrop web interface to get files onto your phone
- Change the save directory for your recieved files
- Start Landrop in the background with Windows
- Keep Landrop running in the System tray
